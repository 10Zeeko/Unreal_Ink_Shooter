# Unreal Ink Shooter - Pol Garriga
## Challenges

- [x] Ink surfaces
- [x] Ink meter
- [x] Different weapons
- [x] Player abilities (Swim and climb while in ink)

## Controls

- Click: Shoots
- WASD: Move the player
- Space bar: Jumps
- Shift: Swims

## Multiplayer
The game can be played with other players at the same network, it uses a lobby system.
